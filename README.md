# 🩺 Diabetes Prediction App

This is a Machine Learning web application built using Flask that predicts whether a person has diabetes or not based on medical inputs.

## 🚀 Features

* User-friendly web interface
* Takes health inputs like glucose, BMI, age, etc.
* Predicts diabetes using trained ML model

## 🛠️ Technologies Used

* Python
* Flask
* Machine Learning (Scikit-learn)
* HTML/CSS

## 📂 Project Structure

* app.py → Main Flask app
* train_model.py → Model training
* model.pkl → Saved ML model
* templates/ → HTML pages
* data.csv → Dataset

## ▶️ How to Run

1. Install requirements
2. Run:
   python app.py
3. Open browser:
  http://192.168.31.39:5000
